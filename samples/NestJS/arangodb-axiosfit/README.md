# ArangoDB Axiosfit Sample

This sample shows how to use Axiosfit in a [NestJS](https://nestjs.com) application calling [ArangoDB](https://www.arangodb.com/) REST API to load the data.

## Install

`npm install`

## Start

`npm start`
